This is a game created for Hack Club's Campfire hackathon, with the theme "beneath the surface." 

You are navigating the crushing darkness of the deep. Use LiDAR flares to illuminate the cavern walls before you collide. The walls reflect neon pink, and coins glitter yellow. Remember to look in every crevice to collect 12 coins and complete the mission!

Gameplay: Use W/S (Up/Down) to propel and A/D (Left/Right) to steer your submarine. Use the space key to blast LiDAR flares. 

Warning: Your submarine can only withstand 3 impacts with the walls.
